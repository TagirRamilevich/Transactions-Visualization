# Transactions-Visualization
A tool for visualizing bank transactions.

To run this dynamic visualization, you will need to convert your data to a file in .tsv format file with 3 columns: time_minutes, category, transactions. 

This repository is a part of this article: https://habr.com/ru/companies/alfa/articles/786524/
